## Description

[Gitlab-Line-Notification](https://github.com/yotewach/gitlab-line-notify) Gitlab Line Notify repository building from nestJS. install guideline

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
